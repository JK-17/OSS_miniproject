## 🙈 miniProject 주제
 - 가계부를 작성해서 용돈 관리 및 지출 금액 관리를 도와 줄 수 있는 프로그램

<br/>

## 🙈 가계부 소개
  - 간단한 입력으로 어떤 카테고리에서 돈을 지출하는지 쉽게 보여줄 수 있는 프로그램 입니다.
  - 금액별로 어떤 지출이 가장 많았는지 부터 순서대로 보여줄 수 있는 프로그램 입니다.
  - 월별 지출 관련해서 상세하게 보여주는 프로그램 입니다.
  - 해당 년도의 총 지출과 수입을 계산해주는 프로그램 입니다.
